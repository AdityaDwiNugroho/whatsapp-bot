24/7 Cloud Hosting Guide

Since this bot is built using Node.js and containerized with Docker, you can run it 24/7 in the cloud so you don't need your laptop to be open.

  • 1
    Hugging Face Spaces (Free): Create a new Space, select the Docker SDK, set it to Private (for privacy), and push this project directory. It builds and runs instantly.
  • 2
    Scan & Link once: Scan the QR code from the Space's web dashboard. The login session is saved in the persistent folder, so it stays connected.
  • 3
    Lock Dashboard: Add the `DASHBOARD_PASSWORD` environment variable in your Space settings to prevent unauthorized access to your dashboard.
